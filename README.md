# NixOS
This is my NixOS configuration repo. 
This configuration used on 2 laptops:

  Lenovo Legion 7 (main and base) - Intel Core i7 10th Gen / 32 GB / 2 TB SSD (LVMed), Nvidia RTX 2080

  Acer Helios Predator 500 - AMD Ryzen 7 2700 / 16 GB / 512 SSD, 1 TB HDD, Radeon RX Vega 56

The configrations based on flake (so one configuration for both laptops), I am using bspwm as WM, themed via pywal.
